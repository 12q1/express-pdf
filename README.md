# express-to-pdf

Generate a PDF file from a web form using Express and NodeJS

## Installing

````
git clone
cd express-to-pdf
yarn
````

## Starting

````
yarn start
````
Visit http://localhost:3000.

## Developing
````
yarn watch
````
